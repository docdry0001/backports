╔══════════════════════════════════════════════════════════════╗
║  INSTALLATION                                                ║
╚══════════════════════════════════════════════════════════════╝

🔥 Method 1: TWRP Recovery (Easiest)
─────────────────────────────────────────────────────────────────
1. adb push wireless-modules-flashable-*.zip /sdcard/
2. Boot to TWRP
3. Flash ZIP
4. Reboot

📱 Method 2: Manual (ADB)
─────────────────────────────────────────────────────────────────
$ adb root && adb remount
$ adb push *.ko /system/lib/modules/
$ adb shell chmod 644 /system/lib/modules/*.ko
$ adb shell insmod /system/lib/modules/compat.ko
$ adb shell insmod /system/lib/modules/cfg80211.ko
$ adb shell insmod /system/lib/modules/mac80211.ko
$ adb shell insmod /system/lib/modules/mt76.ko
$ adb shell insmod /system/lib/modules/mt76-usb.ko
$ adb shell insmod /system/lib/modules/mt7921-common.ko
$ adb shell insmod /system/lib/modules/mt7921u.ko

⚠️  IMPORTANT: Load modules in order (dependencies matter!)

🔍 Run diagnostics first:

$ chmod +x diagnose-build.sh

🐛 Common issues:
─────────────────────────────────────────────────────────────────
Problem: Only compat.ko generated
→ Solution: Use allbackport-fixed.yml or build-wifi-modules.sh

Problem: Symbol errors (ieee80211_*, cfg80211_*)
→ Solution: Run kernel modules_prepare

Problem: CFI/LTO errors
→ Solution: Disable in kernel config (script does this)

Problem: Build fails
→ Check: backports-build.log for errors

📖 Full guide: See TROUBLESHOOTING.md
